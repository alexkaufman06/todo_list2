# To Do List

Lets the user add tasks to a todo list.

## Setup Instructions

### Database Info:
```
Install pg gem
Make a table called list with an id and name column
Make a table called task with id, description, list_id, and due_date columns

```

### Gems Used:
```
sinatra
sinatra-contrib
```
### Install Bundler
```
$ gem install bundler
```
### Run Bundler
```
$ bundle
```
